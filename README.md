# Binance P2P Bot
Binancio calculates median price of any crypto-asset in fiat currency on Binance's p2p exchange right on your console.

## :zap: Usage

```sh
# Using npx
npx binance-p2p-bot

# Using npm
binance-p2p-bot
```

## :cloud: Installation

```sh
# Using npm
npm install -g binance-p2p-bot

# Using yarn
yarn global add binance-p2p-bot
```
